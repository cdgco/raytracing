Sets perspective for output image.

Camera location, subject, viewport, field of view, focus and aperture can be adjusted.

Example: 

Camera(dimensions, Vector3D(5,0,0));

Default values:

  • lookAt: Vector3D(0);
  • vertUp: Vector3D(0, 1, 0);
  • aperture: (double) 0.1;
  • Fov: (double) 40;
